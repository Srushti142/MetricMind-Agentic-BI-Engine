# Data Dictionary

## customers
Contains customer information.

| Column | Description |
|--------|-------------|
| customer_id | Unique customer id |
| customer_city | Customer city |
| customer_state | Customer state |

## orders
Contains order information.

| Column | Description |
|---------|-------------|
| order_id | Unique order id |
| customer_id | Customer reference |
| order_status | Order status |
| order_purchase_timestamp | Purchase date |

## order_items
Contains purchased products for each order.

## products
Contains product information.

## sellers
Contains seller information.

## order_reviews
Contains customer reviews.

## order_payments
Contains payment details.

## geolocation
Contains latitude and longitude information.

## product category name traslation
Contains the name of product category in English.
